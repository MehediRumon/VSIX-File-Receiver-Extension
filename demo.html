File Sender to Visual Studio Demo

Prerequisites:
  • Visual Studio must be running with the File Receiver Extension installed
  • A solution must be open in Visual Studio
  • The extension creates an HTTP server on http://localhost:8080

How to Use the Chrome Extension Content Script

To automatically send files from any web page that generates them, you can dispatch a custom event:

window.dispatchEvent(new CustomEvent('fileGenerated', {
  detail: {
    fileName: 'generated-file.txt',
    content: 'This is the generated content'
  }
}));

API Information

Endpoint: POST http://localhost:8080/

Content-Type: application/json

Request Body:

{
  "fileName": "example.txt",
  "content": "File content here"
}
